# GML_Edu

Clean rebuild repository for MaCAD S.3 — Graph Machine Learning.


Its purpose is to:

* Rebuild the assignment pipeline using the course notebooks and methods.
* Compare workflow structure against selected student repositories for reference.
* Check that graph construction, analysis, and model evaluation meet the course requirements.
* Develop the final project using a consistent residential-plan-to-graph workflow.


## Planned Structure

* `assignment_01_graph_generation` — rebuilt Grasshopper / Rhino / TopologicPy workflow
* `assignment_02_graph_analysis` — rebuilt spatial graph analysis
* `assignment_03_no_grad_evaluation` — rebuilt pretrained-model evaluation
* `final_project` — integrated final project
* `references` — notes comparing course materials and reference repositories
