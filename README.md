# GML_Edu

Clean rebuild repository for MaCAD S.3 — Graph Machine Learning.

This repository reconstructs the full assignment pipeline from first principles, using course notebooks as the authoritative workflow reference and selected student repositories as read-only structural comparisons. No code, geometry, images, or data are copied from external sources.

## Project areas

| Folder | Assignment | Core task |
|---|---|---|
| `assignment_01_graph_generation/` | Assignment 1 | Rhino model → OBJ → TopologicPy → graph → CSV |
| `assignment_02_graph_analysis/` | Assignment 2 | Graph metrics → spatial interpretation → comparison |
| `assignment_03_bgr_evaluation/` | Assignment 3 | BGR Rhino model → four OBJ files → graph creation (13A) → no-grad prediction (13B) |
| `final_project/` | Final project | Comparative residential layouts: two bars vs two L-shaped blocks |

## References

`references/Graph ML -- NOTEBOOKS/` — official course notebooks (read-only).  
`references/course_workflow/` — notes on notebook pipeline and methodology.  
`references/student_repo_comparison/` — structural observations from student repositories (no content copied).
