# Assignment 2 — Graph Analysis

## Purpose

Compute graph metrics on the spatial adjacency graph produced in Assignment 1 and interpret each metric in terms of architectural and spatial meaning.

## Expected inputs

- `nodes.csv` and `edges.csv` from Assignment 1 (or equivalent graph CSV).

## Expected outputs

- `outputs/` — metric tables (CSV or inline notebook results).
- `visuals/` — charts and annotated graph plots illustrating metric distributions.
- `docs/` — written interpretation mapping each metric to a spatial concept (e.g., betweenness → circulation importance).

| Folder | Contents |
