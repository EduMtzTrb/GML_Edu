# Student Repository Comparison

## Purpose

This folder contains structural observations and workflow notes derived from inspecting external student repositories from MaCAD S.3 Graph Machine Learning.

## Ground rules

External student repositories are **read-only references**. The following is strictly prohibited:

- Copying code (Python, notebook cells, scripts)
- Copying geometry (OBJ, 3DM, Rhino files, BREP)
- Copying datasets (CSV, YAML, JSON, PT model files)
- Copying images or visualisations
- Copying written text, comments, or documentation

Only the following is permitted:

- Observing folder organisation and naming conventions
- Noting which notebooks were used and in what order
- Understanding what types of outputs (file formats, naming) were expected
- Identifying pipeline steps that are consistent across multiple students (indicating a course requirement)

## How to use this folder

Add one Markdown note per repository inspected. Each note should record:

1. Repository identifier (no personal names; use a neutral label such as `student_repo_A`).
2. Folder structure observed.
3. Workflow steps inferred from file names and notebook order.
4. Any pipeline details that inform the confirmed or inferred requirements in `REQUIREMENTS_CHECKLIST.md`.

All observations must be paraphrased and abstracted. Do not quote or reproduce any content from external repositories.
